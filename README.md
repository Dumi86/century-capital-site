# Century Capital

Premium digital design studio website built with Tailwind CSS, deployed to Netlify via GitHub.

## Pages

- `/` — Portfolio Overview (Home)
- `/about.html` — About the Firm
- `/contact.html` — Contact Us
- `/services.html` — Marketing & Growth Services
- `/dashboard.html` — Client Dashboard
- `/case-study-aethelgard.html` — Aethelgard Horology Case Study
- `/case-study-centurion.html` — Centurion Mobile Case Study
- `/case-study-monolith.html` — The Monolith Heights Case Study
- `/404.html` — 404 Page

## Tech

- **HTML + Tailwind CSS (CDN)** — No build step needed
- **Fonts** — Newsreader (serif) + Manrope (sans) via Google Fonts
- **Deployed** — Netlify via GitHub auto-deploy

## Deployment

Push to GitHub → Netlify auto-deploys from main branch.
